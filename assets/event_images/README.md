# Hazır Etkinlik Görselleri

Bu klasöre aşağıdaki 5 görseli ekle (kendi telifsiz görsellerin):

- `filter_coffee.jpg`  — Filtre kahve
- `tea.jpg`            — Çay
- `cold_drink.jpg`     — Soğuk içecek
- `turkish_coffee.jpg` — Türk kahvesi
- `other.jpg`          — Diğer (genel)

Öneri: 800x600 px civarı, optimize edilmiş JPEG. Telifsiz kaynak:
Unsplash, Pexels veya kendi çektiğin fotoğraflar.

pubspec.yaml içinde `assets/event_images/` zaten tanımlı.
Görsel eklemezsen uygulama hata vermez; kart ve form bir
yer-tutucu (placeholder) ikon gösterir.
